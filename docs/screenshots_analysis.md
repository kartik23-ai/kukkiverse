## 8. Interactive Device Showcase Blueprint (Live Sandbox)
To display these screenshots on the Live Sandbox page (`sandbox.html`), we will use a **side-by-side premium hardware frame layout**:

1. **Laptop Screen Frame (PC View)**: Left side will display a detailed CSS frame representing a premium physical laptop (bezel, reflection gloss, keyboard base). The active feature's `PC` screenshot is rendered perfectly inside this screen.
2. **Mobile Phone Frame (Mobile View)**: Right side will display a realistic smartphone frame (bezel, curved screen, camera notch, metallic side borders). The active feature's `Mobile` screenshot is rendered inside this screen.
3. **Smooth Cross-fade Switching**: When users click the sidebar controller buttons, the screen images inside BOTH devices will smoothly blur and fade using CSS transitions (`opacity`, `filter: blur()`, and `transform: scale()`).
4. **Three.js Background Morphing**: The 3D canvas background will morph shape and color based on the selected feature (e.g. glowing red wave for *Concert*, slow dark green ring for *Focus*, electric violet/cyan plasma particles for *Labs*).
